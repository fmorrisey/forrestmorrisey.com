## Overview

Spoker v2 is a complete rebuild of the original Spoker application, designed to make sprint planning sessions smooth and engaging for remote teams.

## Key Features

- **Real-time Voting** — WebSocket-powered instant updates
- **Multiple Deck Types** — Fibonacci, T-shirt sizes, and custom decks
- **Session History** — Track and export estimation data
- **Mobile Responsive** — Works seamlessly on any device

## Technical Highlights

Built with Astro for the frontend and a WebSocket server for real-time communication. Docker containers make deployment consistent and reliable.
